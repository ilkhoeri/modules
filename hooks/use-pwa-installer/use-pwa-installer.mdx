$:description
Progressive web apps combine the best features of traditional websites and platform-specific apps.
The assumption is that you have already defined [manifest.json](https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps/Tutorials/js13kGames/Installable_PWAs) in the project, so you want to create a [trigger prompt](https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps/How_to/Trigger_install_prompt) for users to install the PWA from your application.

$:usage
function MyComponent() {
  const { prompt, installer } = usePWAInstaller();

  return (
    <button
      type="button"
      role="button"
      title="install pwa"
      aria-label="install pwa"
      hidden={!prompt}
      onClick={installer}
      className={globalStyle({ button: "default", size: "sm" })}
    >
      Install PWA
    </button>
  );
};
